## これは何か
静岡県東部7市町（函南・伊豆の国・三島・沼津・清水・長泉・裾野）の「再建築できそうな広い物件」を毎日自動で巡回し、**新着／消滅の差分**をレポートにまとめるツール。

---

## 毎日の使い方（3ステップ）
1. **実行**：定期実行を設定済みなら自動。手動なら PowerShell で
   ```
   cd C:\Claude\akiya-watch
   py watch.py
   ```
2. **開く**：`reports\index.html` をブラウザで開く ← これが成果物
3. **見る**：上部や各列のメニューで条件を絞り、気になる物件は「詳細」リンクで元サイトへ

---

## レポートの見方（ざっくり）
- 上から順に：**物件ブラウザ（全件）→ 新着 → NGエリア該当ログ → 消滅**
- セクション見出しをクリックで**開閉**（NGログ・消滅は既定で閉）
- **列の見出しをクリック** → その列の並べ替え／絞り込みメニュー
- 右上で**設定に名前を付けて保存・呼び出し**ができる
- 価格・面積・坪単価のセルは色の濃さで「お得さ」が一目で分かる

---

## 設定ファイル `urls.yaml`（たまに触る）
場所：`C:\Claude\akiya-watch\urls.yaml`

- **`sites:`** … 監視対象サイト。増減は基本 **Claude Code に依頼**（手で書かない）
- **`filters:`** … 画面の「初期値」。普段は画面で調整して保存すればよく、ここを触るのは“毎回開いたときの既定値”を恒久的に変えたいときだけ
  - `price_ceiling_by_type:` … 種別ごとの価格上限（万円）
  - `exclude_areas:` … 通学不可で除外する別荘地名（所在地に含むと除外）
  - 関心／注意キーワードのリスト

> ⚠️ URL自体の追加・変更は、必ず Claude Code に「**HTTP実測して変更案を出して**」と頼む。記憶で書くと存在しないURLが混ざって事故る。
