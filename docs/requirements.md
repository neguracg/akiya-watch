## 1. 背景と目的

### 1-1. 探している物件
静岡県東部6市町（函南町・伊豆の国市・三島市・沼津市・清水町・長泉町）で、**市街化調整区域にある／値段がつきにくいが広い（330㎡以上）／再建築が可能**な不動産。価格1000万円以下が目安。

### 1-2. メインシナリオ
老朽家屋を持て余した所有者から安く買い、建築確認等の申請・建て直しの労力はすべて買主が負う。代わりに安価に入手する。**本質は「土地探し」ではなく「再建築可能な“面倒物件”の発見システム」**。

### 1-3. レビューで判明した本命の所在（重要）
持て余し・低額・ワケあり（調整区域・要セットバック等）物件は、広告費が見合わないため**大手ポータルやREINSにも載らず、地場業者の自社HPや個人直サイトに一次掲載される**ことが多い。良い物件は公開前に地元業者が抜く。したがって**公的バンクは最優先ではない**。

### 1-4. 狙うべきスイートスポット（戦略の核）
「学校近接 × 調整区域 × 広い × 安い」は、青地（農業振興地域内農用地＝農転困難）を引きやすく、原則家が建たない。狙いは『**指定既存集落（都計法34条11号 or 各市町条例12号指定）の白地で、小中学校徒歩圏**』という狭い領域。これは腱の既存のGIS・集落密度・線引き前宅地アプローチが狙う地点と一致する。
※未確定: 「既存宅地確認制度」は廃止済みで現在は34条11号/12号の指定が条件、というレビュー指摘は load-bearing。6市町の現行根拠は一次確認が必要（§11）。

### 1-5. 最終的に作りたい状態
> Web上で見つかる「自分が探している条件の物件」は、もうすべて確認できている。

これを定期実行と新着差分の確認で継続維持する。

## 2. スコープ

### 2-1. 対象
- **対象地域**: 上記6市町
- **対象種別（拡大）**: 土地 / 古家付き土地 / 中古戸建 / 空き家（建物付き含む）／（任意）投資用戸建
- **適合条件（初期値・ユーザーが画面で可変）**: 価格1000万円以下 かつ 面積330㎡以上
- **関心シグナル**: 市街化調整区域・既存宅地・線引き前宅地等（除外せずフラグ）

### 2-2. 自動化の範囲
収集 ＋ 一次フィルタ ＋ 再建築可否の機械ヒューリスティック（参考値）まで。**最終判定は市町窓口確認（§9・人間工程）が必須**。

### 2-3. スコープ外
非公開流通（縁故・相対）。Web監視では原理的に捕捉不能。人的アプローチ（空き家相談会・地場業者訪問）で補う。

## 3. 機能要件

### 3-1. 網羅的監視リストの整備（最重要・継続）
チャネルを6型に再分類し（§4）、6チャネル×6市町×対象種別のマトリクスで「確定/該当なし/未確認」を管理。網羅性＝未確認ゼロ。加えて:
- **漏れ検知ループ**（月1）: 素のWeb検索「{市町} 古家付き土地」「{市町} 市街化調整区域 売地」「{市町} 既存宅地」等で監視リスト外の物件を探し、未知ソースを追加
- **地場業者監視**: 6市町の仲介系地場業者を10〜30社リスト化し、各社の「売地/物件」ページを**ハッシュ監視**（更新検知→人間確認）。Google `site:業者ドメイン {市町} 土地` の定期検索も併用

### 3-2. 各サイトからの物件抽出（site adapter 方式）
サイト個別パーサで物件カード単位に解釈し、**共通スキーマ**へ正規化。未登録サイトは汎用抽出／ハッシュ監視にフォールバック。動的・bot対策サイトはブラウザ自動化（フェーズ2）。

**共通スキーマ（全adapterが埋める目標項目）**:
- site / 市町 / 種別（土地・古家付き・戸建・空き家）/ 所在地 / 価格(万円) / 土地面積(㎡)・建物面積 / 坪単価 / 地目 / 都市計画区分 / 接道(生text) / **上水道(公営/井戸)** / **汚水(下水/浄化槽)** / **ハザード(浸水/土砂イエロー・レッド)** / **温泉引き込み有無** / **指定既存集落該当(要・区域図照合)** / **線引き日(市町別固定値)** / 再建築可否(○△×)＋理由 / フラグ(関心/注意) / 検出日 / 詳細URL
- 取得できない項目は "—"。上下水・ハザード等は掲載に無ければ "要確認" とし、候補化後の確認項目に回す

### 3-3. 一次フィルタ（適合判定）＋クライアント側インタラクティブ絞り込み
- 価格・面積で 適合/数値不明/不適合。ファネル（生→価格取得→面積取得→適合）で精度可視化
- HTMLに全件をJSON埋め込み、ブラウザ上で価格・面積・市町・NGエリアを動かしてリアルタイム絞り込み（既定＝設定値、毎日開くと既定表示、その場の変更はセッション限り）

### 3-4. 除外とフラグ（キーワード戦略）
- **exclude_areas（絶対除外・所在地一致）**: 南箱根ダイヤランド／パサディナタウン／東大場／**伊豆エメラルドタウン／奈古谷系別荘地**（追加随時）。通学が成立しない別荘地。除外ログに「NGエリア:◯◯」と理由付きで残す
- **exclude_keywords は空のまま**（建築可否では機械除外しない）
- **重要原則**: 「要相談／調査中／市街化調整区域」は**除外語にしない**。本命がここに隠れる。すべて下記フラグへ
- **interest_keywords（注目シグナル）**: 既存宅地・線引き前宅地・都市計画法34条11号・34条12号・43条・再建築可・建替可・適合証明・白地・無指定
- **caution_keywords（要注意フラグ）**: 再建築不可・要相談・調査中・農地転用・既存不適格・崖条例・別荘地・管理費・借地・定期借地・青地・農振・温泉

### 3-5. レポート出力（HTML主＋CSV）
- HTML: ①今回適用した条件（1項目1行）②サイト別サマリ（適合・新着＝赤）③インタラクティブ絞り込み＋全件テーブル（坪単価ヒートマップ／再建築理由列／適合行ハイライト／?ヘルプ）④新着 ⑤消滅（7日窓）⑥除外ログ（理由付き）。`noindex` メタ、過去14日ドロップダウン
- CSV: UTF-8 BOM
- ぱっと見でマトリクスとして俯瞰できるコンパクト表。所在地は短縮表示、全文はCSV

### 3-6. 再建築可否ヒューリスティック（参考値・理由必須）
掲載テキストからの推定に過ぎず**法的確定ではない**。判定理由を必ず併記。市街化調整区域は×でなく△（要役場確認＝本命候補）。HTMLに「○△×は推定・市町窓口確認が必要」と明記。

### 3-7. 新着／消滅（二層）
first_seen/last_seen を保持。added＝新着（上部・赤）。removed＝アーカイブ退避し消滅検出から7日掲載。

### 3-8. レポートのナビゲーション
既定＝最新。過去14日を日付ドロップダウンで遡及。静的HTMLのみで成立。

### 3-9. 定期実行（クラウド・PC非依存）※フェーズ3
GitHub Actions の cron で毎日自動実行、GitHub Pages の固定URLで公開（URLを知る誰でも閲覧可・noindex付与・受容済み）。**未検証リスク**: データセンターIPはbot対策でブロックされやすい。移行時に取得可否を実測。

### 3-10. 小中学校までの距離（フェーズ4・実装検討）
住所ジオコーディング→学校位置データ（国土数値情報 等の無償ソース or Google Maps API）→最寄り小中学校距離を算出し列追加。

## 4. チャネル分類（6型・MECE）と優先順位

| 優先 | チャネル | 内容 | 備考 |
|---|---|---|---|
| 1 | ③業者内流通型 | 地場業者の自社HP（・REINS・店頭・看板） | **本命**。低額・ワケあり物件の一次情報。ハッシュ監視＋site:検索 |
| 2 | ④個人掲示板型 | 家いちば・ジモティー・0円/格安系 | 所有者が持て余す層 |
| 3 | ①公開ポータル型 | SUUMO・アットホーム・HOME'S・ハトマーク・不動産ジャパン・ラビーネット・スマイミー静岡 | 市場価格に寄っており異常値は少ないが網羅の土台 |
| 4 | ②公的誘導型 | 空き家バンク（しずおか・LIFULL・アットホーム）・移住サイト・自治体ページ | 信頼は高いが件数少・抜かれやすい・低優先 |
| 5 | ⑤強制換価型 | 競売(BIT)・税公売(KSI/国税)・任意売却・相続財産清算 | 低頻度だが濃い。制度別に監視 |
| 6 | ⑥公有財産売却型 | 国有地・県有地・市町有地の売却 | 強制換価とは別ルート。不要財産の払い下げ |

## 5. 監視サイト・リスト（完全版・ステータス付き）

凡例: **[V]** 実在検証済 / **[C]** 候補（Codeが要HTTP検証）/ **[G]** 生成ルール（Codeが生成＋検証）/ **[要特定]** URL未特定（Codeが探索）

### ① 公開ポータル型（種別: 土地・中古戸建・空き家 を各市町に展開）
- **[G] SUUMO**: 土地＋中古一戸建て。市=`sc_{市町}`、郡=`sc_suntogun`(清水/長泉)・`sc_tagatagun`(函南)＋filter。土地は[V]済、中古戸建URLは[G]
- **[G] アットホーム**: 土地＋中古一戸建て。各市町slug（函南=`tagata_kannami-city`[V]、清水/長泉=`suntou_*`）。bot対策(405)ありフェーズ2可能性
- **[G] LIFULL HOME'S**: 土地＋中古一戸建て。伊豆の国/三島/沼津[一部V]に**函南(`tagata_kannami-city`)・清水(`sunto_shimizu-city`)・長泉(`sunto_nagaizumi-city`)を追加**
- **[C] ハトマークサイト(全宅連)**: 全6市町。行政コード= 函南22325/伊豆の国22225/三島22206/沼津22203/清水22341/長泉22342（長泉[V]、他要確認）。売買 住宅＋土地
- **[C] 不動産ジャパン(不動産流通推進センター)**: `fudousan.or.jp/property/buy/22/area`（22=静岡）＋市町・種別フィルタ
- **[C] ラビーネット不動産(全日)**: `rabbynet.zennichi.or.jp/buy/land/shizuoka`＋市町、及び中古戸建
- **[C] スマイミー静岡(静岡県宅建サポートセンター)**: `shizuoka.fudohsan.jp` 土地/中古住宅の市町検索

### ② 公的誘導型
- **[V] 空き家バンクしずおか**: 6市町。※現URLはハブ。実一覧URL(`/一覧/買う-定住タイプ/地域/{市町}/`)へ要変更案（フェーズ2でadapter化・本命）
- **[V] LIFULL HOME'S 空き家バンク 静岡県一覧**（6市町を文字列抽出）
- **[C] アットホーム空き家バンク 静岡県**（`akiya-athome.jp`、6市町の参画有無を確認）
- **[V] 住むなら三島**（三島市 中古住宅）
- **[V] 長泉町空き家バンク**
- 参照のみ（監視不要）: ゆとりすと静岡（移住ハブ）

### ③ 業者内流通型（最重要・未着手）
- **[要特定] 地場業者の自社HP × 6市町で10〜30社**: 仲介メインの中小業者を選定しリスト化 → 各「売地/物件一覧」ページをハッシュ監視。Google `site:{業者ドメイン} {市町} 土地` も併用
- REINSは一般非公開につき直接監視不可（地場HP・店頭で間接捕捉）

### ④ 個人掲示板型
- **[V] ジモティー 静岡県 土地**（＋不動産カテゴリ）
- **[C] 家いちば 静岡県**（`ieichiba.com/area/shizuoka`）
- **[C]（段階）空き家ゲートウェイ**（100円/格安特化）

### ⑤ 強制換価型
- **[V] BIT 沼津支部**（6市町を管轄）
- **[V] KSI官公庁オークション**（公売）
- **[V] 国税庁公売 不動産**
- **[要特定]（段階）任意売却専門サイト**（士業・相続財産清算経由。実装難・低優先）

### ⑥ 公有財産売却型（必須追加）
- **[C] 静岡県 未利用県有地 売却**（`pref.shizuoka.jp/.../kenyuchi/...`）
- **[要特定] 国有財産売却（東海財務局）**
- **[要特定] 6市町の市有地・町有地売却ページ**（各市HPで個別入札・随契）

### 投資用（任意・段階）
- **[C] 楽待 / 不動産連合隊**: 各市町の収益・築古戸建（古家付きが出る）

## 6. 非機能・運用ルール

- robots.txt の Disallow 遵守。低頻度・サイト間スリープ
- `urls.yaml` が唯一のマスタ。URL変更・追加は (1)Invoke-WebRequest実測＋(2)WebSearch/WebFetch確認 を経た変更案をユーザー承認後にのみ適用。**記憶・推測からURLを生成しない**
- 公開リスク受容済み（GitHub Pages固定URL・noindex付与）
- ドキュメント運用: 本書がマスター。要件更新時は前版＋以降の決定から新バージョン全文を再生成し本人が保存。requirements.md は実行時に読まない（実行時マスタは urls.yaml）

## 9. 必須の人間工程：市町窓口確認フロー

候補化した瞬間に挟むチェックリストをレポートへ自動添付する。**広告の「再建築可」を信じて買うのが最大の落とし穴**。
- [ ] 43条／34条11号・12号の該当可否（市町窓口）
- [ ] 線引き前宅地・既存宅地の扱い（市町の現行基準。三島は線引き日が地区で異なる: 萩2003-12-19/他1972-5-8 ※要確認）
- [ ] 指定既存集落の区域図との照合（大字・字レベル）
- [ ] 青地／白地（農業委員会。農振除外の可否）
- [ ] ハザード（浸水想定・土砂災害イエロー/レッド）→融資・火災保険への影響
- [ ] インフラ（上水道=公営/井戸、汚水=下水/浄化槽、引き込み・解体の概算費）
- [ ] 別荘地・管理組合規約・温泉引き込み権の有無
